AIを活用した在庫最適

2019 Apr 30, 01:25

[etc] : AIを活用した在庫最適

在庫最適、需要予測とは

  • 在庫最適とは？
    • 在庫最適を目的とするために需要予測を行う。
    • 在庫最適の市販ソフト・ERPパッケージのように価格、規模も様々なものが世の中に出回っている
      • 期待需要とは？
    • 「期待需要 = 必要在庫」とは限らない。非常に奥が深い
  • 需要予測とは？
    • 欠品による提供可能だったはずの機会を失うことを防ぐ
    • 在庫が余ってしまうことによる廃棄ロスのリスクを防ぐ
  • MDとは
    • マーチャンダイジング

在庫最適、需要予測で大切なこと

  • 大切なこと
    • どこまで誤差が発生してもビジネスとして成り立つのか。正しく予測をすることは現在の技術では非常に難しいので、予測を外してもカバーできるものが必要
    • 従来は人で行っていたが、人の作業も誤差が非常に大きいので、それよりも良い精度を出すことがゴールとなる
    • よって機械による作業が、人の作業よりも誤差が小さくなることによって、ビジネスのインパクトを出すことが重要
    • 事前に制約事項を決める
      • 需要予測 => 在庫最適をした結果、売上は伸びたが利益が減ったという状況にならないように、事前に利益を一定確保するような制限を入れる
      • 最終的には人間が調整できるような便を用意しておく
    • 徐々に強化してくフィードバックサイクルを回せるか
    • 以下にもともと作業をしていた従業員のフラストレーションを下げられるか
      • AIによる在庫最適を誤ってしまうと作業者の負荷が増えてしまう
    • 欠品率の10%解消で、売上が3〜5倍になるケースも有る
      • ちょっとした工夫で大きな効果が生まれるという側面もある
    • 過去の時系列データから未来を予測すること。大きく方針は下記の3つがある
      • 「傾向」を予測すること。季節や周期などにより傾向を掴むこと
      • 似ている過去のイベントなどから予測。例えば商品販売した時点が一番ピークになるようなもので、そこから徐々に需要が下がっていくことを予測する
      • 外部要因などの前提から予測。例えば気温の変化を読み取って、在庫配置を適切にする
    • アパレルの在庫最適の難しさ
      • そもそもリードタイムが長いので、前年から需要予測 => 在庫最適という形を取らなければならないこともある
      • 最悪、セールとして売り出すことでもある程度の利益確保が見込まれるため、在庫最適の考え方が他の業界と比べると浸透していない状態。製造業などは原価率が高く、在庫最適を誤ると致命的な状態になるので。
    • 需要予測で使われる変数の例
      • 過去の仕入れ実績データ
      • 天気情報
      • 周辺のイベント
      • 人口
      • 地域の所得水準
